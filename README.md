# 🌊 Kumbh Mela Management System

A full-stack MERN web application designed to manage and monitor activities during the Kumbh Mela.
This system helps users and authorities handle crowd management, events, and emergency services efficiently.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* View crowd density (map-based)
* Event viewing
* SOS emergency alert
* Lost & Found services

### 🛠️ Admin / Authority Features

* Add / Delete events
* Monitor crowd levels
* Manage users
* Emergency alert handling

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/ashwini25203/kumb-mela-project.git
cd kumb-mela-project
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```

---


---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```
